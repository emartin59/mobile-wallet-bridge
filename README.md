# kin-mobile-web-bridge


## Server

* make sure you have Nodejs installed
- https://nodejs.org/dist/v10.16.0/node-v10.16.0.pkg

* How to run Socket Backend:
```
cd web/server
npm i
npm run start
```

## Client

* This is the part that mimicks 3ed party like Medium or TapaTalk

```
cd web/client
npm i
npm run start
```
