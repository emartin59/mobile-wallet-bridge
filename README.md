# Kin Mobile-Browser Bridge Solution
- make sure you have Nodejs installed: https://nodejs.org/dist/v10.16.0/node-v10.16.0.pkg

## Server
- Backend with WebSocket
```
cd server
npm install
npm run build
npm run start
```

## Client
- This is the part mimicks 3ed party like Medium or TapaTalk
```
cd client
npm install
npm run build
npm run start
```
